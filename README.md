# Team Task Manager

A simple full-stack web application to manage tasks with role-based access.

## Features
- User Registration & Login
- Role-based access (Admin / Member)
- Create and assign tasks
- Update task status (Pending / Done)
- Simple dashboard

## Tech Stack
- Backend: Flask (Python)
- Database: SQLite
- Frontend: HTML, CSS

## How to Run
1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
